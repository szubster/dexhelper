# DAG Dashboard Visualization & UI

## Overview
This Epic covers the frontend user interface for the DAG Dashboard Webview. It consumes the structured DAG data generated by the data layer and renders it interactively.

## Requirements
- Consume the JSON graph structure provided by the data layer.
- Render a directed graph visualization of the Foundry nodes.
- Display `id`, `type`, `status`, and `owner_persona` within or alongside each node representation.
- Implement interactive filtering controls (e.g., filter by `type` or `status`).
- Implement interactive highlighting (e.g., clicking a node highlights its upstream and downstream dependencies).
- Serve the UI as a local HTML/React webview or integrated admin route.

## Acceptance Criteria
- [ ] Select and integrate a graph rendering library (e.g., Mermaid.js, React Flow).
- [ ] Implement the core graph visualization component.
- [ ] Implement filtering UI controls and logic.
- [ ] Implement dependency highlighting interactions.
- [ ] Wire up the UI to consume data from the parsing layer.
