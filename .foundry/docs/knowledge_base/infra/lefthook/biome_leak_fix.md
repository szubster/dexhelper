Fixed Biome errors bypassing pre-commit hooks. The root cause was a restrictive `glob: "*.{js,ts,jsx,tsx,json,css}"` in `lefthook.yml` which only targeted root files. Switched to a robust `all_files: true` configuration in `lefthook.yml`. This ensures that Biome and TypeScript check all files including subdirectories, bypassing issues with shell expansion and empty argument lists in certain environments.
