Foundry Engine scripts `.github/scripts/foundry-active.ts` and `.github/scripts/foundry-heartbeat.ts` used strict regex for matching `updated_at` timestamps (`/^\d{4}-\d{2}-\d{2}$/`), which failed on full ISO 8601 strings, resulting in duplicated keys or failing tests. Changed the regex to gracefully accept full ISO timestamps or nulls, similarly to how `jules_session_id` is parsed.
