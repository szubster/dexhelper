# Suggested Commands: dexhelper

## Development

- `npm run dev`: Start the development server (port 3000).
- `npm install`: Install project dependencies.

## Testing

- `npm run test`: Run unit tests using Vitest.
- `npm run test:ct`: Run Playwright component tests.
- `npm run test:e2e`: Run Playwright E2E tests.
- `npm run test:e2e:ui`: Open the Playwright Test UI for debugging.

## Verification & Build

- `npm run lint`: Run TypeScript type checking (`tsc --noEmit`).
- `npm run build`: Create a production-ready build.
- `npm run build:analyze`: Build and analyze bundle size using `vite-bundle-analyzer`.
- `npm run preview`: Locally preview the production build.
- `npm run clean`: Remove the `dist/` directory.

## System Utils

- **Linux-based**: `ls`, `grep`, `find`, `git`, `cat`, etc., are available.
