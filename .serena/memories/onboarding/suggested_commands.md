# Suggested Commands: dexhelper

## Development

- `pnpm dev`: Start the development server (port 3000).
- `pnpm install`: Install project dependencies.

## Testing

- `pnpm test`: Run unit tests using Vitest.
- `pnpm test:ct`: Run Playwright component tests.
- `pnpm test:e2e`: Run Playwright E2E tests.
- `pnpm test:e2e:ui`: Open the Playwright Test UI for debugging.

## Verification & Build

- `pnpm lint`: Run TypeScript type checking and style verification.
- `pnpm build`: Create a production-ready build.
- `pnpm build:analyze`: Build and analyze bundle size using `vite-bundle-analyzer`.
- `pnpm preview`: Locally preview the production build.
- `pnpm clean`: Remove the `dist/` directory.

## System Utils

- **Linux-based**: `ls`, `grep`, `find`, `git`, `cat`, etc., are available.
