The project has been migrated to a leaner data model. Pokémon types have been removed from the static generation script (`scripts/generate-pokedata.ts`), the data schema (`src/db/schema.ts`), and all UI components (`src/components/PokemonDetails.tsx`, `src/components/SearchAndFilters.tsx`). The store (`src/store.ts`) no longer supports type-based filtering. The resulting `pokedata.json` is significantly smaller (~687K). All changes were verified with `pnpm type-check` and successfully pass.
